# 🔥 Firewall

> Registry tweak to configure **Windows Firewall** — SharedAccess service, domain/private profiles, Remote Admin, File & Print, RDP, UPnP.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔒 **Firewall Config** | Domain and Private profiles enabled |
| 🚫 **Remote Admin** | Disabled for both profiles |
| 🚫 **File & Print** | Disabled |
| 🚫 **Remote Desktop** | Disabled |
| 🚫 **UPnP** | Disabled |
| 📋 **ConSecRules** | Auth rule configuration |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11, Server |
| **Privileges** | Administrator |

---

## 🚀 Usage

1. **Create a restore point** (recommended)
2. Double-click the `.reg` file or right-click → **Merge**
